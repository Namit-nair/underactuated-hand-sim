# Legacy / archived work

Kept for reference — **superseded by the high-fidelity model** at the repo root
(`config.py`, `analytical_model.py`, `finger_model.py`, `high_fidelity/`).

- **`low_fidelity/`** — the initial low-fidelity finger model and its parameter
  sweeps / validation scripts. This was the first developed version; it predates
  the CAD-accurate geometry and the physics-faithful (steel-string) tendon model.

- **`research_outputs/`** — plots and CSVs produced before the proper geometry
  model existed. Historical record only.

Nothing here is wired into `config.py` and these scripts are not maintained.
They are retained until no longer needed.
